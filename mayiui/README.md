# mayiui

#### 介绍
帮助您快速制作弹窗，消息提示，确认框，Tip框的一个js组件，功能灵活

#### 软件架构
基于umd


#### 安装教程
在页面head区域引入下面文件即可

```
<script type="text/javascript" src="mayiui/mayiui.js"></script>
<link rel="stylesheet" href="mayiui/mayiui.css">
```


#### 使用说明

调用代码简单，比如弹出一个消息只需要使用mayiui.msg('您好');
